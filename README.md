# 📋 Projeto Cadastro

Este é um projeto simples de formulário de cadastro desenvolvido com **HTML**, **CSS** e **JavaScript**.  
O objetivo é praticar conceitos básicos de front-end como estruturação de formulários, estilização com CSS e manipulação de eventos com JavaScript.

---

## 🚀 Funcionalidades
- Formulário com campos para **Nome**, **Email** e **Senha**.
- Validação básica dos campos com `required`.
- Exibição de mensagem de confirmação ao cadastrar.
- Reset automático do formulário após o envio.
- Estilização com CSS (cores suaves, bordas arredondadas e sombra).

---

## 🛠️ Tecnologias utilizadas
- **HTML5** → Estrutura do formulário.
- **CSS3** → Estilização e layout responsivo.
- **JavaScript** → Função de cadastro e interação com o usuário.
